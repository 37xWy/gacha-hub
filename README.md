# 🌠 Astral Gacha Hub
A completely private, client-side web application for tracking gacha pull history.

## Features
* **100% Private:** Built with vanilla HTML/JS. JSON data is parsed locally in your browser and never uploaded to a server.
* **Multi-Game Support:** Automatically formats dashboards for *Honkai: Star Rail* and *Arknights: Endfield*.
* **Analytics:** Tracks 50/50 win rates, streak history, and average pity.
* **Time Machine:** Groups pulls by year for cleaner viewing.

## How to Use
1. Run the Python fetcher script to generate your local `hsr_UID.json` file.
2. Open the GitHub Pages link.
3. Drag and drop your JSON file into the browser.
